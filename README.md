# Delivery_Project

## Overview

배달 어플리케이션 "요기요" PC 버전 Copy

## 개발환경

* Spring Framework
* Java
* MySQL

## 구성
1. 일반 사용자 페이지
   - <http://bendelivery.shop/>
2. 가게 사장님 페이지
   - <http://bendelivery.shop/owner/>
3. 관리자 페이지
   - <http://bendelivery.shop/admin/home>

## 기능

* 일반 사용자
  1. 식당 조회 (카테고리 별)
  2. 식당 찜하기
  3. 주문 장바구니
  4. 주문하기
  5. 리뷰 작성
* 가게 사장님
  1. 가게 운영정보 관리
  2. 메뉴 관리
  3. 리뷰관리
  4. 주문관리
* 관리자
  1. 입점 신청 승인
